# ZERONADE - Sport Bikes Laravel Project

## Project Overview
ZERONADE is a Laravel-based web application showcasing sport bikes. Users can browse products, view details, add them to the cart, and contact the company. The project includes full CRUD functionality, AJAX-powered search, and a map integration for the store location.

---

## Features

### User Features
- Browse the bike catalog with images, prices, and details.
- Dynamic product pages using slugs.
- Add products to cart and manage quantities.
- Contact form for inquiries.
- Map integration showing the store location using Leaflet.js.

### Admin Features
- Manage products: Create, Read, Update, Delete (CRUD).
- Manage members (limited CRUD functionality).
- AJAX search functionality for products.
- Highlight searched products in the admin panel.

### Technical Features
- Laravel framework (PHP 8+, MVC structure)
- MySQL database
- Blade templating engine
- AJAX integration for live search
- Leaflet.js API for map visualization
